# flutter_example

这里是我测试flutter插件的例子，根据个人使用会陆续更新，配套有文章和视频的介绍。

项目使用fvm做了虚拟环境，没有fvm环境的可以删除.fvm文件夹，和.vscode中的setting配置。

已演示的插件内容：
- Intl国家化功能
- Intl数字和日期格式化功能
- Xupdate一键更新


### 运行环境：
```
Flutter 3.0.1
Dart 2.17.1
DevTools 2.12.2
```
